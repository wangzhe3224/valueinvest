## 一、相对估值法进阶变体

### 1.1 Shiller PE / CAPE (周期调整市盈率)

| 项目 | 内容 |
|------|------|
| **核心公式** | CAPE = 股价 / (10年平均通胀调整后收益) |
| **原理** | 消除商业周期对收益的影响，使用10年平滑数据 |
| **适用场景** | 市场整体估值、指数估值、周期股 |
| **局限性** | ① 对个股适用性有限 ② 不适用于高成长公司 ③ 会计准则变化影响可比性 |
| **实践者** | Robert Shiller (诺贝尔经济学奖) |
| **当前标普500 CAPE** | ~39.7 (历史均值17.3，偏高) |

**使用建议**：可作为市场整体估值模块，辅助判断买入时机。

### 1.2 EV/EBITDA 进阶变体

| 变体 | 公式 | 特点 |
|------|------|------|
| **EV/EBIT** | EV / 息税前利润 | 排除折旧摊销影响，适合资本密集行业 |
| **EV/Invested Capital** | EV / 投资资本 | 衡量资本效率 |
| **EV/Sales** | EV / 销售额 | 适用于亏损但高增长公司 |

### 1.3 PEG 扩展

| 方法 | 公式 | 说明 |
|------|------|------|
| **PEG** | P/E ÷ 增长率(%) | < 1 低估，1-2 合理，> 2 高估 |
| **PEGY** | P/E ÷ (增长率 + 股息率) | 加入股息因素 (Peter Lynch) |

**Peter Lynch 规则**：
- PEG < 1: 可能被低估
- PEG = 1-2: 公允估值
- PEG > 2: 可能被高估

## 二、资产价值法

### 2.1 NAV (净资产价值) - REITs专用

| 项目 | 内容 |
|------|------|
| **核心公式** | NAV = (资产市场价值 - 负债) / 股数 |
| **原理** | 房地产按市场价值重估，而非账面价值 |
| **适用场景** | REITs、房地产基金、基础设施公司 |
| **局限性** | ① 依赖第三方评估 ② 流动性折扣难以量化 |
| **关键指标** | P/NAV 比率 (价格与NAV的比值) |

**REITs额外指标**：

| 指标 | 公式 | 说明 |
|------|------|------|
| **FFO** | 净利润 + 折旧 - 资产出售损益 | 核心运营现金流 |
| **AFFO** | FFO - 资本性支出 - 租赁摊销 | 更精确的现金流 |

### 2.2 清算价值 (Liquidation Value)

| 项目 | 内容 |
|------|------|
| **核心公式** | 清算价值 = 快速变现资产 - 所有负债 - 清算成本 |
| **资产折扣率** | 现金100% \| 应收80-90% \| 存货50-70% \| 固定资产10-50% |
| **适用场景** | 困境公司、破产重组、特殊机会投资 |
| **实践者** | Seth Klarman (Baupost Group) |

### 2.3 重置成本 (Replacement Cost)

| 项目 | 内容 |
|------|------|
| **核心公式** | 重置成本 = 重新构建公司资产所需的总成本 |
| **适用场景** | 制造业和重资产行业 |
| **局限性** | 难以精确计算，新技术可能使旧资产贬值 |

### 2.4 Tobin's Q

| 项目 | 内容 |
|------|------|
| **核心公式** | Q = 企业市场价值 / 资产重置成本 |
| **解读** | Q < 1: 市场价值低于重置成本，可能低估<br>Q > 1: 市场价值高于重置成本，可能高估 |
| **适用场景** | 资本密集型行业、制造业 |
| **局限性** | 无形资产难以估算重置成本 |

## 三、价值投资大师方法

### 3.1 Warren Buffett 所有者盈余 (Owner Earnings)

| 项目 | 内容 |
|------|------|
| **核心公式** | 所有者盈余 = 净利润 + 折旧摊销 - 维护性资本支出 - 营运资本变化 |
| **来源** | 1986年伯克希尔年报 |
| **与FCF关系** | 本质是调整后的自由现金流，更强调"维护性"资本支出 |
| **适用场景** | 所有产生现金流的企业 |
| **实践要点** | 维护性资本支出 vs 增长性资本支出的区分是关键 |

```python
# 伪代码示例
owner_earnings = (
    net_income 
    + depreciation 
    + amortization 
    - maintenance_capex  # 关键：仅维护性，不含扩张性
    - delta_working_capital
)
```

**Buffett 估值原则**：
> "内在价值是企业在剩余生命周期中可以提取的现金的折现值。"

### 3.2 Peter Lynch 方法

| 方法 | 公式 | 说明 |
|------|------|------|
| **PEG扩展** | PEG = P/E / (增长率 × 股息率调整) | 加入股息因素 |
| **增长率调整** | P/E应等于长期增长率 | P/E 15x 对应 15% 增长 |
| **分类方法** | 六类公司分类 | 慢速、稳定、快速、周期、资产、困境 |

**Lynch 六类公司**：
1. **Slow Growers**: 增长 < 6%，高分红
2. **Stalwarts**: 增长 6-12%，蓝筹
3. **Fast Growers**: 增长 12-20%，PEG关键
4. **Cyclicals**: 周期股，需判断周期位置
5. **Turnarounds**: 困境反转，高风险高回报
6. **Asset Plays**: 资产价值被忽视

**历史业绩**：Fidelity Magellan Fund (1977-1990) 年化收益 29.2%

### 3.3 Seth Klarman 安全边际

| 项目 | 内容 |
|------|------|
| **核心理念** | 买入价格低于内在价值的30-50% |
| **计算方式** | 安全边际 = (内在价值 - 买入价) / 内在价值 |
| **最低要求** | 通常要求 ≥ 30% |
| **适用方法** | NCAV、清算价值、保守DCF |

**Klarman 10条投资规则**：
1. 绝对估值优先于相对估值
2. 安全边际至关重要
3. 关注风险，不仅仅是回报
4. 独立思考，避免羊群效应
5. 市场无效提供机会
6. 特殊机会的系统性分析
7. 时间是朋友（长期持有）
8. 现金流是根本
9. 理解你买什么
10. 准备偏离市场

### 3.4 Howard Marks 周期定位

| 项目 | 内容 |
|------|------|
| **核心理念** | "你无法预测，但你可以准备" |
| **定量指标** | P/E、P/B、Shiller CAPE、利率、信用利差、VIX |
| **定性指标** | 投资者情绪、IPO活动、媒体报道 |
| **实践** | Oaktree Capital 定期备忘录 |

**周期阶段定位**：

| 阶段 | 特征 | 投资策略 |
|------|------|----------|
| 早期复苏 | 估值低、情绪悲观 | 激进买入 |
| 中期扩张 | 估值合理、增长健康 | 标准配置 |
| 后期狂热 | 估值高、情绪乐观 | 谨慎/减持 |
| 崩溃期 | 估值下降、恐慌 | 寻找机会 |

## 四、现代/量化方法

### 4.1 EVA (经济增加值)

| 项目 | 内容 |
|------|------|
| **核心公式** | EVA = NOPAT - (投资资本 × WACC) |
| **解读** | 企业创造的超过资本成本的价值 |
| **估值应用** | 企业价值 = 账面价值 + Σ(EVA折现值) |
| **适用场景** | 所有企业，特别是资本效率关键的企业 |
| **实践者** | Stern Stewart & Co. |

**与残差收益模型的关系**：数学上等价，EVA侧重企业层面，RI侧重股权层面。

### 4.2 CFROI (现金流投资回报率)

| 项目 | 内容 |
|------|------|
| **核心公式** | CFROI = 调整后现金流 / 总投资 |
| **特点** | ① 通胀调整 ② 使用经济折旧而非会计折旧 ③ 基于IRR原理 |
| **适用场景** | 跨期比较、跨国比较、资本密集行业 |
| **实践者** | Holt Value Associates (现为Credit Suisse) |
| **局限性** | 计算复杂，数据要求高 |

### 4.3 实物期权估值 (Real Options)

| 项目 | 内容 |
|------|------|
| **核心思想** | 将金融期权定价理论应用于实物投资决策 |
| **期权类型** | 扩张期权、放弃期权、延迟期权、转换期权 |
| **定价模型** | Black-Scholes、二叉树、蒙特卡洛模拟 |
| **适用场景** | ① 研发型公司 ② 采矿/石油储备 ③ 专利/知识产权 |
| **局限性** | 参数估计复杂、需要专业建模能力 |
| **实践案例** | 波音公司用于评估新机型开发 |

### 4.4 残差收益模型 (Residual Income)

| 项目 | 内容 |
|------|------|
| **核心公式** | RI = 净利润 - (股权成本 × 账面价值) |
| **估值** | 价值 = 账面价值 + Σ(RI折现值) |
| **适用场景** | 银行、保险、账面价值重要的行业 |
| **与DCF关系** | 数学上等价（假设干净盈余） |

> 注：项目已在 bank.py 中实现了 Residual Income 方法。

## 五、特定行业估值

### 5.1 保险行业内含价值 (Embedded Value)

| 项目 | 内容 |
|------|------|
| **核心公式** | EV = NAV + VIF (现有业务价值) |
| **NAV** | 调整后净资产价值 |
| **VIF** | 现有保单未来利润的现值 |
| **变体** | TEV、EEV、MCEV (市场一致内含价值) |
| **适用场景** | 寿险公司、产险公司 |

### 5.2 SaaS/订阅制估值

| 指标 | 公式 | 说明 |
|------|------|------|
| **ARR** | 年度经常性收入 | 核心收入指标 |
| **MRR** | 月度经常性收入 | ARR / 12 |
| **LTV** | ARPU × 毛利率 × 平均客户生命周期 | 客户终身价值 |
| **CAC** | 获客总成本 / 新客户数 | 客户获取成本 |
| **LTV/CAC** | LTV ÷ CAC | >3 健康，>5 优秀 |
| **Rule of 40** | 增长率% + FCF利润率% | ≥40 健康 |
| **Burn Multiple** | 净燃烧额 / 新增ARR | <2 优秀 |

**SaaS估值倍数参考**：

| ARR规模 | 估值倍数 |
|---------|----------|
| < $5M | 4-8x ARR |
| $5-20M | 6-12x ARR |
| > $20M | 8-20x ARR (取决于增长率和利润) |

### 5.3 商品/资源股估值

| 方法 | 公式 | 适用 |
|------|------|------|
| **NPV (净现值)** | Σ(CF_t / (1+r)^t) - 初始投资 | 矿山、油田项目 |
| **储备估值** | 探明储量 × 预期价格 - 开采成本 | 石油、天然气 |
| **资源股周期调整** | 使用周期中点收益计算 | 大宗商品公司 |

## 六、周期股估值

### Bernstein 周期调整法

| 项目 | 内容 |
|------|------|
| **核心思想** | 使用完整周期(5-10年)的平均收益，而非单年度 |
| **公式** | 正常化P/E = 股价 / 周期平均EPS |
| **适用场景** | 汽车、钢铁、化工、航运等强周期行业 |
| **替代方案** | 使用EBITDA或FCF替代EPS |

**周期股投资要点**：
- 在周期低点买入（高P/E，低利润）
- 在周期高点卖出（低P/E，高利润）
- 关注产能利用率和库存周期

## 七、学术/前沿方法

### 7.1 机器学习辅助估值

| 方法 | 应用 |
|------|------|
| **回归模型** | 基于历史数据预测公允价值 |
| **聚类分析** | 识别可比公司 |
| **异常检测** | 发现错误定价机会 |
| **NLP情感分析** | 新闻、财报情感 → 估值调整 |

**挑战**：
- 数据质量要求高
- 模型可解释性差
- 过拟合风险
- 市场环境变化导致模型失效

### 7.2 宏观因子调整

| 因子 | 影响 |
|------|------|
| **利率** | 折现率调整 |
| **通胀** | 实际收益率调整 |
| **GDP增长** | 行业增长率调整 |
| **汇率** | 跨境公司估值 |

**调整公式**：
```
调整后估值 = 基础估值 × 宏观因子调整系数
WACC_调整 = WACC_基础 + 宏观风险溢价
```

## 九、关键公式速查表

```python
# ============ 所有者盈余 ============
Owner Earnings = Net Income + D&A - Maintenance CapEx - ΔWorking Capital

# ============ Shiller PE (CAPE) ============
CAPE = Price / (10年平均通胀调整后EPS)

# ============ EVA ============
EVA = NOPAT - (Invested Capital × WACC)

# ============ NAV (REITs) ============
NAV = (市场价值资产 - 负债) / 股数
FFO = Net Income + Depreciation - Gains on Sales
AFFO = FFO - CapEx - Lease Amortization

# ============ SaaS关键指标 ============
LTV = ARPU × Gross Margin × Average Lifespan
CAC Payback = CAC / (ARPU × Gross Margin)
Rule of 40 = Growth % + FCF Margin %
Burn Multiple = Net Burn / Net New ARR

# ============ 残差收益 ============
Value = Book Value + Σ(RI / (1+r)^t)
RI = Book Value × (ROE - COE)

# ============ 周期调整 ============
Normalized P/E = Price / (周期平均EPS)

# ============ 清算价值 ============
Liquidation Value = Σ(资产 × 变现折扣) - 负债 - 清算成本

# ============ Tobin's Q ============
Q = Market Value / Replacement Cost
```
